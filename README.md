# Phishing-detection-dashboard
Cybersecurity dashboard that detects phishing URLs and visualizes risk using charts and attack maps.

## Features

- URL phishing detection
- Risk score visualization
- AI URL analyzer
- Cybersecurity dashboard
- Threat analytics charts
- Live cyber attack map
- Scan history
- Phishing awareness education

## Technologies Used

Frontend
- HTML
- CSS
- JavaScript

Libraries
- Chart.js
- Leaflet.js

Tools
- Visual Studio Code
- GitHub

## How to Run

1. Download the repository
2. Open the folder
3. Open index.html in your browser

## Project Demo

This project simulates phishing detection by analyzing:

- HTTPS usage
- Suspicious keywords
- IP based URLs
- Long URLs
- Subdomain abuse
